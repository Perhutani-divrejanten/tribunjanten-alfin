Logo Tribun Janten Preview

Cara Convert ke PNG:

  1. Metode 1 (Paling Mudah):
    • Buka browser dan kunjungi: Convertio.co atau CloudConvert.com
    • Upload file: img/logo-tribunjanten.svg
    • Pilih format output: PNG
    • Download file PNG-nya
  2. Metode 2 (Screenshot):
    • Print halaman HTML ini dengan Ctrl+P
    • Pilih "Save as PDF" atau print ke PDF
    • Atau langsung screenshot logo dengan Print Screen
    • Crop gambarnya dan simpan sebagai PNG
  3. Metode 3 (Jika punya ImageMagick):
    • Buka Command Prompt di folder img/
    • Jalankan: magick logo-tribunjanten.svg -density 150 -background none logo.png

" File SVG tersimpan di: img/logo-tribunjanten.svg
